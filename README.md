 Da Vinci: The Portraitist (Digital Exhibit)
 A 9-page static website created as a digital exhibit and study guide for the most famous portraits and anatomical studies by Leonardo da Vinci.

 This project is a non-commercial, educational static website built using HTML5 and CSS for a Mark Lewis. 
 The exhibit provides structured analytical content and historical context for each featured artwork.
 Key Features9-Page Structure:
 A complete, navigable exhibit.Structured Analysis: Each portrait page contains dedicated sections for History & Provenance, Technique, and Subject & Interpretation.
 Consistent Navigation: 
 Full navigation is present on every page, allowing seamless movement between the 8 featured works.
 Simple & Accessible Design: 
 Utilizes simple CSS layouts (using floats) for a clean, classic aesthetic suitable for an art history exhibit.
